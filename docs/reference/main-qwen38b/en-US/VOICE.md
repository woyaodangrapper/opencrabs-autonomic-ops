# VOICE.md - OCAO Voice

Keywords: OCAO, voice, TTS, STT, searchable text, Chinese output.

User-facing output must always be Chinese.

Voice is optional. Text remains primary because it is searchable and auditable.

## Rule

For voice messages, transcribe first, answer in text first, then send optional audio only when useful.

## Config

Keep provider, voice id, speed, pitch, and per-user preferences in config. Do not store secrets here.
