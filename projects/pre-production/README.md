# Qwen3:8B 小模型任务原子层

| 功能模块 | 8B 负责的行为 | 主要输入 | 主要输出 | 可靠边界 |
|---|---|---|---|---|
| 日志处理 | 去噪、去重、切分、多行合并、格式归一、字段抽取 | 原始日志流 | 标准化事件 | 适合高频批处理 |
| 事件归并 | 语义近似合并、同类异常聚类、重复告警折叠 | 多条相似日志/告警 | 单一事件簇 | 只能做候选归并 |
| 摘要压缩 | 长日志压缩、时间窗口摘要、关键信息提炼 | 大段文本/Trace | 短摘要 | 适合上下文压缩 |
| 分类标注 | Error Type、Service、Severity、Component 标注 | 事件文本 | 结构化标签 | 类别必须有限 |
| 配置解析 | YAML / ENV / Compose / JSON 结构提取 | 配置文件 | 配置要点 | 适合格式固定输入 |
| 依赖抽取 | Service、Port、Upstream、Downstream、Env 关系提取 | 配置 / 拓扑 / 清单 | 依赖关系表 | 不适合大图推理 |
| Trace 解释 | Span 摘要、延迟段定位、异常链路提炼 | OpenTelemetry Trace | 关键链路摘要 | 只做局部解释 |
| Runbook 填充 | 将模板补全为标准步骤 | 故障描述 | 运维文档草案 | 不做最终执行决策 |
| 命令生成 | 生成 grep、kubectl、docker、journalctl 等命令 | 自然语言 | 可执行命令 | 需白名单约束 |
| 工单结构化 | 自然语言转 JSON / 字段对象 | Issue / Chat / Ticket | 工单对象 | 适合表单化 |
| 路由前判定 | 判断是否需要升级大模型 | 请求文本 + 上下文 | 路由建议 | 仅做分流 |
| 上下文压缩 | Token 压缩、冗余折叠、关键信号保留 | 长上下文 | 短上下文 | 对后续模型最有价值 |

# Qwen3:8B 系统功能模块层

| 功能模块 | 包含的统一行为 | 8B 的角色 | 大模型 / 规则系统角色 |
|---|---|---|---|
| 文本处理 | 清洗、去噪、摘要、结构化、分类、聚合 | 主力执行器 | 验收与兜底 |
| 事件处理 | 日志、告警、异常归并与标注 | 事件归类器 | 最终优先级与处置策略 |
| 拓扑理解 | 配置、依赖、服务关系抽取 | 关系提取器 | 跨服务影响分析 |
| 诊断辅助 | 线索提炼、候选原因、排障建议 | 初筛器 | 根因确认 |
| 操作生成 | Runbook、命令、工单、报告生成 | 文本生成器 | 审批与执行控制 |
| 路由控制 | 简单任务直出、复杂任务升级 | 复杂度判定器 | 最终模型分派 |
| 上下文管理 | 长文本裁剪、关键信号保留、记忆压缩 | 压缩器 | 策略与存储 |
